PureNutrition Supplements Store

Опис

**PureNutrition** е веб апликација за продажба на спортски суплементи. Апликацијата овозможува корисниците да ги прегледуваат, додаваат, уредуваат и бришат суплементите. Оваа апликација е изградена користејќи Node.js, Express, MongoDB и EJS.

Инсталација

1. Клонирајте го репозиториумот:
    ```bash
    git clone https://github.com/your-username/purenutrition.git
    ```
2. Навигирајте до проектната папка:
    ```bash
    cd purenutrition
    ```
3. Инсталирајте ги потребните пакети:
    ```bash
    npm install
    ```

Конфигурација

1. Уверете се дека MongoDB серверот е стартуван локално или на облак. Ако користите локален MongoDB, уверете се дека работи на `mongodb://localhost:27017/supplements`.
2. Во проектната папка, креирајте `.env` датотека и додадете го следниов ред за да го конфигурирате URL за MongoDB конекцијата:
    ```plaintext
    MONGO_URI=mongodb://localhost:27017/supplements
    ```

Структура на проектот

- `app.js` - Главниот влезен фајл за апликацијата
- `models/` - Содржи модели дефинирани со Mongoose
    - `supplement.js` - Моделот за суплементите
- `routes/` - Содржи сите рутери за апликацијата
    - `index.js` - Главниот рутер
    - `supplements.js` - Рутерот за управување со суплементите
- `views/` - Содржи EJS шаблоните
    - `index.ejs` - Почетната страница
    - `add-supplement.ejs` - Формата за додавање нов суплемент
    - `edit-supplement.ejs` - Формата за уредување на суплемент
    - `list.ejs` - Листата на суплементи
    - `login.ejs` - Формата за најава
    - `register.ejs` - Формата за регистрација
- `public/` - Содржи статички фајлови како CSS, слики и видеа

Стартување на апликацијата

1. Стартувајте го серверот:
    ```bash
    npm start
    ```
2. Отворете го вашиот прелистувач и одете на http://localhost:3000 за да ја прегледате апликацијата.

## Употреба

Најава и регистрација

- Корисниците можат да се регистрираат на `/register` и да се најавуваат на `/login`.

Управување со суплементи

- Прегледување на сите суплементи на `/supplements`.
- Додавање на нов суплемент на `/add-supplement`.
- Уредување на суплемент на `/supplements/edit/:id`.
- Бришење на суплемент (ќе треба да додадете функционалност во шаблоните за оваа операција).
